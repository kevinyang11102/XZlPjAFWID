# 蛋糕甜品店管理系统 java514

## 项目概述

本项目是基于`SpringBoot`与`Vue`框架开发的在线蛋糕商城管理系统。项目采用前后端分离的架构模式，代码规范，注释详尽，便于学习和理解。系统功能全面，覆盖了在线商城前后台应有的各项功能。

## 技术栈

- **后端**: SpringBoot
- **前端**: VueCli
- **数据库**: MySQL
- **持久层框架**: MyBatis
- **UI框架**: ElementUI
- **运行环境**: IntelliJ IDEA2019.3.5, MySQL5.7, JDK1.8, Maven3.6.3, Node14.16.1

## 功能模块

### 系统角色

- **普通用户**: 登录注册、商品浏览、购物车管理、订单结算、个人信息管理、订单详情查看等。
- **管理员**: 商品信息管理、订单信息管理、用户信息管理、收益数据图表查看等。
